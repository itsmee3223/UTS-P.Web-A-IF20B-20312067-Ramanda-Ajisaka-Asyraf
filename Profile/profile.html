About

Nama saya Ramanda Ajisaka Asyraf sering dipanggil "Rama" atau "Aji" hobi saya membaca, menoton film, dan mengoding. Saya lahir di Bandar Lampung tanggal 29 oktober 2001. Awal mula saya mengenal tentang programming adalah ketika saya masih SD kelas 3, saat itu saya diajak teman saya untuk bermain warnet. Saya mulai fokus mempelajari bahasa pemrograman waktu saya masuk SMK karena kebetulan dijurusan saya yaitu jurusan Elektronika terdapat mata pelajaran pemrograman.Bahasa pemrograman pertama yang saya pelajari waktu SMK adalah Visual Basic dan juga bahasa C. Waktu belajar tersebut saya disuruh untuk membuat

sebuah program untuk mengontrol LED agar tampil nama saya. Semenjak dari situ saya semakin tertarik tentang dunia teknlogi yang begitu keren dan canggih. Setelah lulus dari SMK saya sempat mendapat tawaran beasiswa di Unila dari kepala jurusan saya, cuman saya menolaknya karena ingin segera bekerja untuk membantu orang tua saya. Selama bekerja 9 bulan saya mendapatkan informasi dari teman yang berkuliah di Teknokrat tentang beasiswa kuliah, akhirnya saya mendaftar dan alhamdulillah diterima. Akhirnya saya dapat kembali melanjutkan cita-cita saya untuk menjadi programmer dan berkuliah di Teknokrat jurusan Infromatika
